src/app/core/interceptors/auth.interceptor.ts

authInterceptorFn
Type : HttpInterceptorFn
Default value : (req, next) => { if (req.url.startsWith('/assets/') || req.url.endsWith('.json')) { return next(req); } const authService = inject(AuthService); const token = authService.getToken(); const headers: Record<string, string> = {}; if (token) { headers['Authorization'] = `Bearer ${token}`; } const clonedRequest = req.clone({ withCredentials: true, setHeaders: headers, }); return next(clonedRequest); }

Añade Authorization: Bearer y withCredentials en peticiones al API. En producción (front y API en dominios distintos) el Bearer es imprescindible; las cookies httpOnly del API no siempre viajan en cross-site.

src/app/core/configs/validators.ts

DNI_NIE_PATTERN
Type : unknown
Default value : /^([0-9]{8}[A-Z]|[XYZ][0-9]{7}[A-Z])$/

Expresión regular unificada para la validación de DNI y NIE en España.

  • DNI: 8 dígitos seguidos de una letra.
  • NIE: Una letra inicial (X, Y, Z), 7 dígitos y una letra final.
