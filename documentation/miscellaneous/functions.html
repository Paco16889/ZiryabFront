isTeacherAssignmentRowSchedulable
isTeacherAssignmentRowSchedulable(row: TeacherSubjectAssignmentRow)

Comprueba si una fila de asignación docente puede usarse al montar horarios (está “schedulable”).

  • Solo se acepta explícitamente estado ACTIVE como impartible.
Parameters :

timeRangesOverlap
timeRangesOverlap(aStart: string, aEnd: string, bStart: string, bEnd: string)

Indica si dos intervalos de tiempo del mismo día se solapan (hay al menos un minuto en común).

Cada intervalo es [inicio, fin] en minutos derivados de HH:mm. Dos clases consecutivas 09:00–10:00 y 10:00–11:00 no solapan (el fin de una es el inicio de la otra).

Parameters :
Name Type Optional Description
aStart string No

Inicio del primer intervalo HH:mm.
aEnd string No

Fin del primer intervalo HH:mm.
bStart string No

Inicio del segundo intervalo HH:mm.
bEnd string No

Fin del segundo intervalo HH:mm.
Returns : boolean

true si hay solape; si no, false.

numberToPrismaDayOfWeek
numberToPrismaDayOfWeek(day: number)

Convierte el día de la UI (1–7) al literal que espera el backend al crear o actualizar horarios.

  • 1 → MONDAY, …, 7 → SUNDAY.
  • Valores fuera de 1–7 devuelven MONDAY por defecto.
Parameters :
Name Type Optional Description
day number No

Día en formato numérico 1 = lunes … 7 = domingo.
Returns : string

String en mayúsculas compatible con Prisma / API.
prismaDayOfWeekToNumber
prismaDayOfWeekToNumber(day)

Convierte el valor de día que viene del API o de Prisma al número que usa el front.

  • Si ya es un número entre 1 y 7, se devuelve igual (no se modifica).
  • Si es un string tipo MONDAY, se mapea a 1, etc.
  • Si el string no es reconocido, se devuelve 1 (lunes) por defecto.
Parameters :
Name Optional Description
day No

Día como string Prisma (MONDAY…) o número 1–7.
Returns : number

Entero 1–7 (lunes = 1, domingo = 7).
